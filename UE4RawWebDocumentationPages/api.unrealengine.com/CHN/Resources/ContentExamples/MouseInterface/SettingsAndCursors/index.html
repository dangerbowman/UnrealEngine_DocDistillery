鼠标互动属性

本页面的内容：
  • 概述
  • 鼠标接口属性

概述

下文是设置鼠标互动以及可用指针示例时使用的可用属性细分。 请注意显示的指针将会基于不同的操作系统而变化。

鼠标接口属性

在玩家控制器类中，您将可以找到名称为 Mouse Interface （鼠标接口）的属性分类。 这就是对基于指针的鼠标互动进行各项设置的地方。 属性如下所示：

属性

描述

Show Mouse Cursor (显示鼠标指针）

该属性判定鼠标指针是否可见。 请注意默认情况下鼠标指针直接使用操作系统的指针。

Enable Click Events （启用点击事件）

该属性使点击事件得以处理。 不设置这个属性，您将无法点击物体。

Enable Mouse Over Events （启用鼠标悬停事件）

该属性使悬停事件得以处理。 不设置这个属性，鼠标悬停无法正常运行。

Default Mouse Cursor （默认鼠标指针）

该属性设置了鼠标指针的默认外观。 此选项可以在可用指针 部分中找到。

Default Click Trace Channel （默认点击追溯通道）

该属性设置了追溯系统将要使用的默认通道，以查看您点击了哪些物体。 默认设置为 Visibiltiy （可见性）。 这就表示追溯会检查物体的可见度状态并仅在物体可见的情况下才返回true值。 我们换一种说法，也就是说您仅能与在游戏中 可见 的物体进行互动。

可用指针

下方列表包含了所有可用指针，您一般在Windows的操作系统中可以见到它们。 不过，您不应由于它们的通常作用而感到拘束。 您可以按您所需，在游戏中使用任意指针。 但是您需要注意，许多电脑用户习惯于在非常特定的情况下看到这些指针，如果您不按照这些规则，可能会导致他们的困惑。

这次，鼠标指针直接照搬自操作系统。 以下指针是在使用Aero主题的Windows中截屏得到的。 在其它操作系统或主题中，鼠标的外观会有所不同。

设置

描述

示例

None （无）

此设置会有效地隐藏指针。 然而此时依然要注意，因为指针的位置仍在被计算，所以依然会有悬停和点击事件。

N/A

Default （默认）

该设置下为默认的箭头指针。

Text Edit Beam (文本编辑符）

当文本区域可被编辑时，一般会出现该指针。

Resize Left Right (水平调整）

该指针出现在任意可进行水平调整的窗口或UI资源的边缘处。

Resize Up Down (垂直调整）

该指针出现在任意可进行垂直调整的窗口或UI资源的边缘处。

Resize South East (沿东南斜线方向调整）

该指针出现在任意可从右上角或右下角进行水平和垂直调整的窗口或UI资源的边缘处。

Resize South West (沿西南斜线方向调整）

该指针出现在任意可从左上角或左下角进行水平和垂直调整的窗口或UI资源的边缘处。

Cardinal Cross (十字）

当您从屏幕上拖动窗口时常能看到该指针。

Crosshairs （准心）

在执行不同的精确操作时，会出现该指针。

Hand （手）

当您在超链接上悬停鼠标时，一般会出现这个指针。

Grab Hand （握手型）

当您的鼠标停留在屏幕中试图拖动的物体上时，会出现该指针。

Grab Hand Closed （握紧手型）

当您使用鼠标在屏幕中拖动物体时，一般会出现这个指针。

Slashed Circle (不可用）

当您尝试执行非法操作，例如将某项目拖动到其无法到达的位置时，会出现该指针。

Eye Dropper （滴管）

当您对给定像素位置进行颜色取样时，最常见该指针。
