创建GPU粒子效果

Beginner
Unreal Engine 4.21

先决条件主题

本页假定您已对以下主题有一定的了解。请先阅读这些主题，然后继续进行操作。

  • 启用Niagara插件NEW!

要获得某些效果，您可能需要生成数万个粒子。然而，使用标准的CPU生成这么多粒子会导致游戏变慢。在以下教程中，我们将演示如何使用GPU而非CPU生成粒子。

步骤

  1. 首先，在 内容浏览器（Content Browser） 中右键单击，创建一个Niagara发射器，并在显示的菜单中选择 FX > Niagara发射器（FX > Niagara Emitter）。将显示Niagara发射器（Niagara Emitter）向导。

  2. 单击 从发射器模板创建新发射器（Create a new emitter from an emitter template） 旁边的单选按钮。选择 喷泉（Fountain） 模板。然后单击 确定（OK） 以在内容浏览器（Content Browser）中创建新Niagara发射器。

  3. 将新建的Niagara发射器命名为 FX_GPU，然后双击打开。

  4. 展开 发射器生成（Emitter Spawn） 部分。找到 发射器属性（Emitter Properties） 并展开。单击 模拟目标（Sim Target） 旁边的下拉菜单。从显示的列表中，选择 GPUCompute Sim 选项。

此设置将粒子模拟从CPU更改为GPU。这使得引擎能够在保持性能的同时产生更多的粒子。

6.展开 发射器更新（Emitter Update） 部分。找到 生成率（Spawn Rate） 模块。

  1. 在生成率（Spawn Rate）模块中，将 生成率（SpawnRate） 的值设置为 5000。

  2. 下一步，找到 设置变量（Set Variables） 模块。将 Particles.SpriteSize X 和 Y 值设置为 5。

  3. 在 球体位置（Sphere Location） 模块中，将 球体半径（Sphere Radius） 的值设置为 100。

  4. 单击 编译（Compile）、应用（Apply） 和 保存（Save） 按钮以编译并保存您的作业。然后关闭 FX_GPU 发射器。

  5. 在 内容浏览器（Content Browser） 中右键单击，并从显示的菜单中选择 FX > Niagara系统（FX > Niagara System）。将显示Niagara系统（Niagara System）向导。

  6. 单击 创建一个不带发射器的空系统（Create an empty system with no emitters） 旁边的单选按钮。然后单击 确定（OK） 以在内容浏览器（Content Browser）中创建新Niagara系统。

  7. 将新的Niagara系统命名为 GPU。 

  8. 双击 GPU Niagara系统（GPU Niagara System） 以打开它，然后将 FX_GPU Niagara发射器从内容浏览器拖动到 时间轴（Timeline）。

  9. 单击 编译（Compile） 和 保存（Save） 按钮以编译并保存您的作业，然后关闭GPU Niagara系统。在 内容浏览器（Content Browser） 中，将 GPU Niagara系统（GPU Niagara System） 拖动到关卡视口中。完成后，应可看到如下图所示的一些内容。
