主菜单栏

Unreal Engine 4.9

任何使用过Windows应用程序的人都应当会很熟悉编辑器中的 菜单栏（Menu Bar）。它提供了对编辑器中处理关卡时所用通用工具和命令的访问权限。

文件

命令

说明

加载并保存（Load and Save）

新关卡（New Level）

创建新关卡，或选择要从其开始着手的关卡模板。

打开关卡（Open Level）

加载一个现有关卡。

保存（Save）

将当前关卡保存到磁盘。

另存为（Save As）

用给定名称保存当前永久性地图。

保存所有关卡（Save All Levels）

保存所有未保存的已加载关卡，包括流送关卡。

打开资源（Open Asset）

召唤一个资源选取器。

全部保存（Save All）

保存所有未保存的关卡和资源。

选择要保存的文件（Choose Files to Save）

打开一个带关卡和内容的保存选项的对话框。

提交到源控件（Submit to Source Control）

打开一个带内容和关卡相关选取选项的对话框。

项目（Project）

新项目（New Project）

打开一个对话框以创建新的游戏项目。

打开项目（Open Project）

打开一个对话框以选择要打开的游戏项目。

将代码添加到项目（Add Code To Project）

将C++代码添加到项目。只有安装了Visual Studio，才能编译代码。

烘焙内容（Cook Content）

为特定平台烘焙项目内容。如果您在Visual Studio中运行基于代码的项目，此命令将非常有用。

打包项目（Package Project）

编译、烘焙和打包您的项目及其内容，以便分发。

打开Visual Studio（Open Visual Studio）

在Visual Studio中打开您的C++代码。

刷新Visual Studio项目（Refresh Visual Studio Project）

在Visual Studio中刷新您的C++代码项目。

Actor

导入（Import）

将对象和Actor从T3D格式导入到当前关卡。

全部导出（Export All）

将整个关卡导出到磁盘上的文件（支持多种格式）。

导出选定项（Export Selected）

将当前选择的对象导出到磁盘上的文件（支持多种格式）。

应用程序（Application）

将关卡添加到收藏夹（Add Level To Favorites）

设置当前加载的关卡是否会出现在收藏夹关卡列表中。

最近的关卡（Recent Levels）

选择要加载的关卡。

最近的项目（Recent Projects）

选择要切换至的项目。

退出（Exit）

退出应用程序。

编辑

命令

说明

历史记录（History）

撤销（Undo）

撤销上一次完成的操作。

重做（Redo）

重做上一次撤销的操作。

编辑（Edit）

剪切（Cut）

剪切选定项。

复制（Copy）

复制选定项。

粘贴（Paste）

粘贴剪贴板内容。

重复（Duplicate）

重复选定项。

删除（Delete）

删除当前选择。

配置（Configuration）

编辑器首选项（Editor Preferences）

配置此编辑器的行为和功能。有关详细信息，请参阅 编辑器偏好设置 。

项目设置（Project Settings）

更改当前加载项目的设置。有关详细信息，请参阅 项目设置 。

插件（Plugins）

此命令将打开插件浏览器，可以在其中为当前项目启用或禁用这些插件。

窗口

命令

说明

细节（Details）

显示 细节（Details） 面板，该面板使您能够访问所选对象的各种属性。最多可以同时显示4个 细节（Details） 面板。

视口（Viewports）

显示视口，即您在游戏三维世界中的视图。最多可以同时显示4个视口。

层级LOD大纲视图（Hierarchical LOD Outliner）

此命令将显示HLOD设置并允许生成群集。

图层（Layers）

打开 图层（Layers） 面板，您可以在其中将您的关卡划分为一系列的可见层。

关卡（Levels）

打开显示永久性关卡和所有流送关卡的 关卡（Levels） 面板。

模式（Modes）

显示 模式（Modes） 面板，您可从该面板访问特殊的关卡编辑工具，例如地形（Landscape）和植物（Foliage）。

世界场景大纲视图（World Outliner）

打开 世界场景大纲视图（World Outliner），即关卡中当前所有Actor的层级明细。

统计数据（Statistics）

显示了 统计数据（Statistics） 面板，该面板使您能够访问关卡的各种性能统计数据。

工具栏（Toolbar）

显示工具栏（Toolbar），其中包含在关卡编辑期间将执行的各种常见功能。

世界场景设置（World Settings）

显示 世界场景设置（World Settings） 面板，可在其中查找特定于您当前正在编辑的关卡的设置。

内容浏览器（Content Browser）

打开最多4个单独的内容浏览器（Content Browsers） 以处理内容资源。

开发人员工具（Developer Tools）

使得使用引擎和编辑器的程序员能够访问各种有用的 开发者工具 。

蓝图调试器（Blueprint Debugger）

打开 蓝图调试 面板。

类查看器（Class Viewer）

打开 类查看器 以将 Actor 实例添加到地图并创建 蓝图（Blueprints）。

消息日志（Message Log）

打开 消息日志（Message Log），其中包含各种专业信息日志。任何给定时间都仅存在一个 消息日志（Message Log）。

输出日志（Output Log）

打开一个新的 输出日志（Output Log）。任何给定时间都仅存在一个 输出日志（Output Log）。

插件（Plugins）

打开插件浏览器（Plugins Browser）选项卡。

虚幻前端（Unreal Frontend）

设备管理器（Device Manager）

打开 设备管理器（Device Manager） 选项卡。

会话前端（Session Frontend）

打开 Unreal Frontend(虚幻前端) 工具，此工具使您能够访问自动化系统（Automation System） 、虚幻控制台等。

布局（Layout）

重置布局（Reset Layout）

备份用户设置并重置布局自定义。

保存布局（Save Layout）

保存当前用户界面布局。

启用全屏（Enable Fullscreen）

在全屏模式和普通模式之间切换编辑器。

帮助

命令

说明

浏览（Browse）

文档（Documentation）

打开主文档页面。

API参考（API Reference）

打开API参考文档。

视口控件（Viewport Controls）

打开视口控件备忘单。

教程（Tutorials）

打开介绍教程，它们将介绍有关如何使用虚幻引擎4编辑器的基本知识。

在线（Online）

包含Epic Games和虚幻引擎支持站点的链接。

应用程序（Application）

关于虚幻编辑器（About Unreal Editor）

打开关于（About）对话框，显示有关引擎版本的信息。
