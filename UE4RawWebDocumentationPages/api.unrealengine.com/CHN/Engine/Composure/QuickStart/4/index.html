4 - Composure 关卡序列设置

Beginner
Unreal Engine 4.17
Previous Step
Next Step
Composure 快速入门

上一部分中我们创建了 Composure 所必需的关卡序列。下一部分中我们将对此关卡序列进行设置，使其在播放时与 Composure 兼容。

步骤

  1. 首先点击绿色的 +Track 按钮并从出现的列表中选择 Camera Cut，在 Sequencer 标签中添加一个新的 Camera Cut 轨道。

  2. 在 World Outliner 上选择 Cine Camera Actor，然后将其从 World Outliner 拖至 Sequencer 窗口，添加到时间轴。

    点击查看全图。

    在之后数个步骤中进行内容添加时需要确保时间轴位于 1098 帧上，否则内容将无法正确匹配。

  3. 然后在 Camera Cut 轨道中按下 +Camera 按钮，并选中之前被添加到 Sequencer 的电影摄像机 Actor。

  4. 接下来在 World Outliner 中选中以下蓝图并将它们添加到 Sequencer 轨道中。完成操作后，轨道的视图与下图类似：

    • BP_AdvancedCompositingSkeletalMesh

    • BP_AdvancedCompositing Blueprints

  5. 在轨道列表中的 BP_AdvancedCompositingSkeletalMesh 蓝图下按下 Animation 旁的 加号，然后从显示的列表中选择 mocap_patio_walk 动画。

  6. 确保时间轴位于第 1098 帧上，然后在 BP_AdvancedCompositingSkeletalMesh 的 Transform 部分将 Location 和 Rotation 设为如下数值：

    • Location:-20, -140, 0

    • Rotation:0, 0, 5

    点击查看全图。

  7. 接下来在轨道列表中点击 CineCameraActor1，然后从菜单中选择 Import 选项。然后在显示的窗口中导航至本入门指南的第一部分中下载的 shot0030_camera_export.fbx 文件，将其选中后按下 Open 键导入。

  8. 在 Camera Cut 轨道中点击 Camera 图标，通过摄像机进行查看；然后拖动时间轴查看摄像机和人体模型之间的相互动作。

  9. 现在我们需要添加以下 Composure 特定的输入：点击 BP_AdvancedCompositing 蓝图右侧的 加号，并从列表中选择以下项目：

    • Input 0

    • Input 1

    • Distortion Displacement Map

      添加图像序列后 Distortion Displacement Map 轨道将为空白，因为 Image Plate 资源的帧率设为 0。

    • Overscan Sensor Width

    • Overscan Sensor Height

    • Compositing Mode

    • Shadow Color Grading > Gain

    点击查看全图。

  10. 添加所需的 Composure 轨道后，将时间轴设为 1098 帧，然后按下每个轨道旁的 加号 输入选择数据、或在数字输入中进行输入，将以下数据输入：

    • Input 0 shot0030_background

    • Input 1 shot0030_matte

    • Distortion Displacement Map shot0030_distortion

    • Overscan Sensor Width 37.795

    • Overscan Sensor Height 21.238

    • Compositing Mode Shadow + Occlusion

    • Shadow Color Grading > Gain X:-0.28, Y:-0.35 Z:-0.25

    点击查看全图。

最终结果

设置好 Sequencer 和 Composure 的兼容后，下一部分将设置所需的游戏模式和关卡蓝图逻辑，按下 Play 或 Simulate 键后即可将所有内容串联起来。
