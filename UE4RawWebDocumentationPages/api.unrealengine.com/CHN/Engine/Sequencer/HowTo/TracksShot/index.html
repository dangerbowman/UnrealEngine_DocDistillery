创建镜头和副镜头

Intermediate
Unreal Engine 4.12

与传统电影编辑器类似，镜头轨道 允许您添加、排列和重排关卡序列，这些关卡序列将按排列顺序播放。 添加的每个关卡序列都称为 镜头，而每个镜头可以有多个 副镜头。通过副镜头，可以将镜头快速切换为替代版本，这使您能够快速体验过场动画的不同效果。

在本操作指南中，我们将使用镜头轨道创建一段多摄像机过场动画，并为一个镜头创建多个副镜头。

步骤

在本操作指南中，我们使用 Blueprint Third Person Template 项目，并启用了 起步内容。

  1. 在 内容浏览器 中，创建一个文件夹和四个 关卡序列，并对它们进行重命名，如下所示。

    它们分别命名为 Master、Shot01、Shot02 和 Shot03。

  2. 打开 Shot01，并单击 添加摄像机（Add Camera）按钮。

  3. 将视口中的摄像机移到角色的脚部前方。

    虚幻引擎新增内容请参见 视口控件 页面，了解导航方法。

  4. 在 Sequencer 中，在第 0 帧处添加用于 转换（Transform）的键，并调整 手动焦距（Manual Focus Distance），然后添加键。

    示例中使用的手动焦距值为 123，但是，您可以随意使用期望的设置。

  5. 擦除至序列末端，向上移动 CineCamera 使其朝向角色面部，然后为其 转换（Transform）添加一个键。

    这样，当序列播放时，摄像机将从脚部向上移到头部。

  6. 打开 Shot02，并重复添加摄像机移动效果的过程。

    在这里，我们将摄像机放在角色脚部左侧，然后随着时间推移使其向上移。您还可以调整 手动焦距（Manual Focus Distance）并为您的更改添加一个键，以使焦点落在角色上。

    要重温添加摄像机移动的方法，请参见 使用镜头切换 。

  7. 打开 Shot03，并为此镜头添加摄像机移动。

    在这里，我们将摄像机移至角色脚部右侧，调整焦点，然后使其随着时间推移向上移动至头部。

  8. 在 内容浏览器**中，打开 Master 关卡序列，然后添加 镜头轨道**。

  9. 单击 添加镜头（Add Shot）按钮，然后添加 Shot01。

  10. 通过单击 添加镜头（Add Shot）按钮，添加 Shot02 和 Shot03。

  11. 左键单击 镜头并将其拖动到新的位置可重排顺序。

    在上图中，我们先播放原始设置，然后通过重排镜头顺序并在序列中移动镜头进行快速更改设置。

  12. 右键单击 某个镜头，并选择 新建副镜头（New Take）选项。

    系统将显示保存确认，并自动为该镜头指定保存名称/位置。

    您可以在 项目设置（Project Settings）的 关卡序列（Level Sequences）下调整镜头/副镜头的命名约定。

  13. 右键单击 新的副镜头，并选择 新建副镜头（New Take）来创建另一个副镜头。

  14. 右键单击 新的副镜头，并突出显示 副镜头（Takes）。

    每个副镜头都将显示在 副镜头（Takes）菜单中，您可以通过从该菜单中选择不同的副镜头来快速更换镜头。 副镜头是复制的关卡序列，这意味着，您对它们进行更改并不会更改原始序列，但会更改复制的版本。

    目前在第一次创建副镜头时，在菜单中它会作为“副镜头 2”（Take 2）列出，而“副镜头 1”（原镜头）不会列出，但在 内容浏览器 中仍然存在。为了解决这个问题，您可以创建一个副镜头（作为原镜头使用），然后再创建一个用来修改的副镜头（这样您就可以使用“副镜头”（Take）菜单在它们之间快速切换）。

最终结果

现在您可以使用 镜头轨道 排列您的镜头，并快速切换有不同副镜头的镜头。

以上示例显示如何使用镜头轨道来快速修改镜头播放顺序，以及如何使用“副镜头”系统切换镜头。借助副镜头，您可随意试验镜头的播放，因为通过切换到原始副镜头，可以快速恢复这些复制的镜头。
