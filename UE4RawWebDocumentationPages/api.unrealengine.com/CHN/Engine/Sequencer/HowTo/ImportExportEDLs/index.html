导入和导出编辑决策列表（EDL）

Advanced
Unreal Engine 4.13

虚幻引擎 4 的 Sequencer 不但能渲染并导出整个动画，还可按每个镜头导入动画，并包含一个 编辑决策列表（或称作 EDL）。该文件可在多数视频编辑软件中使用，如 Premiere、Avid，或 Final Cut。

EDL 包含卷轴和时间码数据的排序列表，代表为生成最终剪辑而获取每个视频片段的位置。 在此指南中，我们导出了一个动画和 EDL 文件，将其用于其他软件（如 Premiere）、对其进行编辑，并将编辑后的动画随修改一同重新导入虚幻引擎。

步骤

在此指南中我们使用 Blueprint Third Person Template 项目并创建了一个动画范本，您也可使用自建项目和动画序列。 此指南使用 Adobe Premiere Pro CS6 作为外部编辑软件，您可以使用支持 EDL 导入的任意软件。

  1. 确定需要导入的动画含 Shot Track（内含所有镜头），然后点击 Render Movie 按钮。

  2. 使用 {shot} 标记定义 Output Directory 和 Filename Format，并勾选 Write Edit Decision List。在此显示需要随 EDL 导出的动画样本，然后点击 Render Movie 按钮呼出设置对话窗口。在设置窗口中新建动画导出的文件夹，并将动画的命名作为前缀添加到 {shot} 标记。使用 {shot} 标记后，每个镜头将随整体动画导出为单独的视频文件。

    Output Format 设为 Video Sequence 方有效。对单帧无效。

  1. 在软件编辑套件中 导入 上一步中导出的 EDL 文件。在 Premiere Pro 中，可在窗口左下角 点击右键 并选择 Import，然后指向 EDL 文件。 在 EDL Information 和 New Sequence 对话窗口中留下默认信息。

  2. 下一步需要将 EDL 的数据链接到编辑软件中的导出媒体文件。在 Premiere Pro 中 右键点击 media 文件夹并选择 Link Media，然后将每个文件指向其相应的媒体文件。

  3. 根据需要在编辑软件中修改镜头，然后作为 EDL 导出。我们调整场景样本中镜头的长度，并重新排列它们在序列中的位置。

  4. 返回 虚幻引擎 4，在序列中 右键单击 Shot Track，然后选择 Import EDL。在其他软件中进行的修改便会反映到虚幻引擎 4 的序列中。

最终结果

Sequencer 中的导入/导出 EDL 选项便于您灵活决定在何处进行编辑。

您可在 Sequencer 中完成大部分设置，然后将序列传入外部应用程序（如 Premiere）中的编辑器进行最终修改，然后再传回 Sequencer 进行调整。

将序列作为 EDL 导出并在外部进行修改后，即可在最终动画的渲染前添加后期加工效果。
