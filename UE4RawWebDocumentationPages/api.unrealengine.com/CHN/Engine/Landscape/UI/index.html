地形UI参考

Unreal Engine 4.9

此文档包含 地形 系统和编辑器所有图标的参考。

管理

此模式可用于新建地形、从现有地形添加或移除分段、移动地形，并将样条添加到地形。

新建地形

使用“新建地形”类型中的参数来新建一个地形。

选择

选择一个地形分段，实现部分属性（如材质和光照属性）的逐分段覆盖。

添加

将一个地形分段添加到当前地形。或基于笔刷尺寸中的值添加多个分段。

删除

移除地形的一个分段。与可视性工具不同的是，此操作将移除地形数据的分段，完全删除数据。

移动关卡

将选中的组件移动到指定的流送关卡。

修改组件大小

根据设置修改组件和地形的大小。

编辑样条

使用此工具时，在地形上 Ctrl + 左键点击 将放置一个样条的节点。如一个节点已选中，另一个节点已放置，则此工具将把新节点连接到之前的节点。如未选中节点，则将放置一个新的初始节点。

造型

使用此工具能对地形高度图的外观进行造型，满足项目需求。

造型工具

升高和降低笔刷影响范围中的地形。用 鼠标左键 可升高地形，用 Shift + 鼠标左键 可降低地形。

平滑工具

拉平地形顶点的Z轴位置，平滑笔刷影响范围中的地形。

平整工具

此工具将升高降低其影响范围中的地形，使其与开始使用此工具时所处位置的Z轴高度相同。

斜坡工具

此工具将在两个特定的点之间创建一个斜坡，根据设置添加衰减。注意：此工具无法使用任何笔刷。

侵蚀工具

侵蚀工具利用热力侵蚀模拟调整高度图的高度。它模拟土壤从高处向低处移动的效果。高低落差越大，产生的侵蚀效果越强。该工具还可在侵蚀上应用噪点效果，呈现出自然真实的随机外貌。

水力侵蚀

水力侵蚀工具利用水力侵蚀模拟调整高度图的高度。将使用一个噪点过滤器来决定初始降雨的出现位置。计算出的模拟将确定初始降雨的水流，以及消融、水流迁移和蒸发。计算结果将生成用于降低高度图的实际数值。

噪点工具

噪点工具将噪点过滤器应用至高度图或图层权重。强度决定噪点的量。

拓扑

拓扑与平滑工具相似，它将推拉三角形来平滑地貌的过渡，然而其将尝试保持地貌的基础形式，最小化Z轴方向的变化。X/Y偏移贴图会使地形渲染变慢，并会以其他工具进行绘制，因此建议尽量使用拓扑工具。

可视性

结合地形洞穴材质 使用时，此工具可遮挡地形区域的可视性和碰撞。

选择

使用者可在地形上放置一张遮罩，对造型或绘制进行更为精确的控制；使地形分段无法被编辑。

复制/粘贴

复制和粘贴可从地形的一个区域复制地貌数据，并将其粘贴到地形的另一个区域。甚至可以保存复制粘贴数据，用于其他地形中。

镜像工具

此工具可将地形的一侧复制到另一侧，以便轻松地镜像或旋转地形几何体。

绘制

使用者可选择性地将不同材质应用到地形的不同部分，以便修改地形的外观。

绘制工具

绘制工具可增加或减少应用到地形的材质层的权重。

平滑工具

平滑地形材质图层的权重，在不同区域之间创建平滑的过渡。

平整工具

抓取此工具的初始位置，然后将此图层权重应用到笔划的位置。

噪点工具

噪点工具将噪点过滤器应用至高度图或图层权重。强度决定噪点的量。

笔刷

这部分可修改对地形进行绘制和造型时所使用笔刷的外观和效果。

圆圈笔刷

这是一个简单的圆形笔刷。使用的衰减大于0时，用此笔刷进行绘制或造型将创建更为柔和的边缘。

透明度笔刷

用笔刷笔划安置遮罩图像的方向。使用者可从选为遮罩的图像中选择使用的通道。

图案笔刷

此笔刷将在整个地貌上排下重复的遮罩。

组件笔刷

此笔刷可上下移动整个地形组件。笔刷的大小将定义同时受影响的分段的数量。

小工具笔刷

小工具笔刷将定义进行复制和粘贴的区域。与传统复制/粘贴不同的是，使用者可将从.raw和.r16文件中导入的数据粘贴到小工具中。

笔刷衰减

该部分可修改笔刷的衰减方式。

平滑衰减

拥有良好引入引出的衰减。

线性衰减

一个常量衰减。

球形衰减

此衰减起初较缓，在靠近笔刷边缘时迅速结束。

尖端衰减

此衰减迅速下坠，然后趋于平缓，梯度轻微而平缓。要求精度和一定过度绘制时使用最佳。
