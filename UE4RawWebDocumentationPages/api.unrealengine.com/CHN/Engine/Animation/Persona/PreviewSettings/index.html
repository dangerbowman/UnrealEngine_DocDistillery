预览场景设置

预览场景设置（Preview Scene Settings） 面板使用户无需在场景中进行设置即可迅速了解资源在多个环境和光照情景下的效果。在编辑器内每个 动画工具 中，能对从骨架网格体到灯光和后期处理效果等不同设置进行定义和应用。

Preview Scene Settings 位于所使用的 动画工具 的右上角中。

关闭 Preview Scene Settings 后，可在工具栏的 Window 选项中重新启用。

也可从 Viewport Options 下拉菜单中访问 Preview Scene Setting 面板。

动画

此部分用于定义将动画应用到预览网格体的方法。

选项

描述

Animation Mode

实现预览动画的方法（Default、Reference Pose 或 Use Specific Animation）。

Animation

使用的预览动画。

模型

可在此部分修改用于预览的骨架网格体。

选项

描述

Preview Mesh (Animation)

用于预览的骨架网格体。

每类资源，无论是动画序列、动画蓝图，或者骨架均可保存，并用不同的预览网格体覆盖。 覆盖顺序从骨架到资源，因此如果资源不包含覆盖，其默认为骨架的预览网格体。

额外网格体

可在此将额外的骨架网格体作为预览网格体集合的一部分纳入到预览场景中。

选项

描述

Additional Meshes

用作预览所含网格体的预览网格体集合。

Skeletal Meshes

特定预览网格体集合中所包括的骨架网格体资源。

有数个骨架网格体相连时（如一个角色的组件），预览网格体集合十分实用。 此外，预览网格体集合灵活性极高，支持在未保存的集合中使用未保存资源。 需要和其他系统接合，用其他骨架网格体部分“打造”一个骨架网格体的项目也可对预览网格体集合进行延展。

设置

这部分包含定义预览视口组成的诸多设置。

选项

描述

Profile Name

用于分辨配置文件的命名。

Shared Profile

此配置文件是否应该保存在 project.ini 文件中。

Directional Light Intensity

手动设置定向光照强度（0.0 - 20.0）。

Direction Light Color

手动设置定向光照颜色。

Sky Light Intensity

手动设置天空光照强度（0.0 - 20.0）。

Rotate Sky and Directional Lighting

开启/关闭天空和定向光照的旋转。按 K 键拖动手动旋转天空光照，旋转定向光照则按 L 键。

Show Environment

开启/关闭环境球体的可视性。

Show Floor

开启/关闭地面网格体的可视性。

Environment Cube Map

设置用于天空光照和反射的环境立方体贴图。

Post Processing Enabled

后期处理是否影响场景。

Lighting Rig Rotation

天空当前的旋转值（0 - 360 度）。

Rotation Speed

开启旋转后天空旋转的速度。

用户的设置可被保存为 配置文件，可通过窗口底部的 Profiles 菜单添加或在配置文件之间切换。

颜色分级

此部分包含视口中的诸多 颜色分级（Color Grading） 选项。

请在 颜色分级和胶片色调映射器 中查阅详细内容。

透镜

可在此部分将诸多后期处理效果添加到视口。

请在 后处理效果 中查阅详细内容。

渲染功能

可在此部分将诸多光照效果添加到视口中。

请在 照亮环境 中查阅详细内容。

色调映射器

在这部分中可对诸多 色调映射 功能进行调整。

请在 Tone Mapping 中查阅详细内容。

移动色调映射器

可在此部分定义移动平台的 色调映射 功能。

请在 移动平台上的后期处理效果 中查阅详细内容。
