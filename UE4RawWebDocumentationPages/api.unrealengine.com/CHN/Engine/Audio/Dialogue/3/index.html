使用对话语音和和Waves

Dialogue Voices 和 Dialogue Wave 设置完成后，即可对任务 NPC 设定逻辑，创建正确的打招呼语境。 此外，对话语境（Dialogue Context） 包含至少两个 Dialogue Voices、一个 Speaker 和至少一个对话为 Directed At 的 Dialogue Voice。

  1. 将一个 Box 组件添加到 Mixamo_SWAT 蓝图。

  2. 在 视口 中调整 Box 组件，使其高度与角色身高相仿，并与角色保持一定距离。不要让它和角色产生重叠，或对设置角色的胶囊体组件不生成重叠事件，否则角色将自行触发对话逻辑。

  3. 新增名为 Greeting 的 Dialogue Wave 变量。

  4. 编译蓝图，并将之前创建的 Dialogue Wave 的默认值设为 QuestGreeting。

  5. 设置以下蓝图图表。
