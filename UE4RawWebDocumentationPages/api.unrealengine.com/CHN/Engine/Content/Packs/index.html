初学者内容

Unreal Engine 4.9

当您根据模板来创建新项目时，您可选择将初学者内容包括在项目中。这会将所有那些内容都复制到您的项目中，这样您就会有 自己的个人副本可以使用，并可以迅速开始建立游戏构想的原型。

要将初学者内容包括在项目中，请确保在“新建项目”（New Project）向导中启用“包含初学者内容”（With Starter Content）。如果当前设置为“无初学者内容”（No Starter Content），请单击“初学者内容”（Starter Content）框，然后选择 “包含初学者内容”（With Starter Content）。

“初学者内容”包中包含各种可帮助您完成设计的资产。这些资产包括材质、粒子效果和基本形状，甚至还有几个可用来确保将项目设置为 正确比例的道具。而且，许多 视频教程 和 快速入门指南 使用初学者内容作为要处理的资源。

将初学者内容添加到项目之后，您可以在 内容浏览器 中的“初学者内容”（StarterContent）文件夹中找到此内容。

要查看使用中的初学者内容，请打开 起步贴图（Starter Map） 关卡。在这里，您可以看到此关卡内的各个位置使用了许多粒子系统、材质和静态网格。

高级照明贴图

在初学者内容中，还有一个 Advanced_Lighting 贴图，它设置为使美工可以快速方便地在理想照明条件下查看其创作成果。

打开 Advanced_Lighting 贴图之后，您就可以开始将资源放入 SM_AssetPlatform 阶段。但是，大量的设置工作由 BP_LightStage 蓝图完成。 这个蓝图的行为就像是定向光线、天空盒球体、HDRI 背景、大气、高度雾、天空光和光束的总成。它使美工可以在一个位置完成大部分 照明设置。在关卡中旋转 BP_LightStage Actor 将定义太阳方向和 HDRI 旋转。大气也会对 Actor 的旋转作出反应。

您可以对 BP_LightStage 设置许多属性 - 或者设置为蓝图的默认值，或者针对放置在关卡中的实例进行设置。

属性

效果

全局亮度（Global Brightness）

针对所有亮度值调整全局乘数。

使用 HDRI（Use HDRI）

是否使用 HDRI 作为背景。如果使用 HDRI，那么还有用于设置亮度、对比度、色调、立方体贴图和旋转的其他设置。

如果要将 HDRI 与定向光搭配使用，那么您可能希望导入前先在 HDRI 中呈现太阳，然后使太阳方向与 HDRI 中的太阳一致。拍摄参考对象可帮助您使直接光与间接光的比率相符。

使用太阳光（Use Sun Light）

启用默认情况下可移动的定向光。

太阳光静止（Stationary Light for Sun）

使定向光静止。

覆盖太阳颜色（Override Sun Color）

默认情况下，太阳从与大气散射相符的角度获取其颜色。此选项设置为 true 时，将禁用此行为，而改为使用“色调”（Tint）。

禁用太阳光晕（Disable Sun Disk）

使定向光不在大气中产生太阳光晕。

阴影距离（Shadow Distance）

设置绘制动态阴影的距离。

使用大气（Use Atmosphere）

启用大气散射。BP_LightStage 的大部分大气设置对应于大气雾 Actor 的相同设置。

使用雾（Use Fog）

启用指数高度雾。

海拔高度（Altitude）

设置雾的高度，以使您不必移动 Actor。

光束高光（Light Shaft Bloom）和 光束遮蔽（Light Shaft Occlusion）

切换这些属性将启用或禁用太阳方向所产生的后处理光束。

要在现有项目中使用高级照明资源，您可 迁移 Advanced_Lighting 贴图，这将同时迁移所有相关联的资源。
